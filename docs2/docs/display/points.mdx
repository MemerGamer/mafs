# Points

Points can be rendered at a location <em className="font-display">(x, y)</em>. Plain old `Point`s cannot be draggedâ€”for that, see [Movable points](/interaction/movable-points).

```jsx live
function SimplePoint() {
  return (
    <Mafs height={500}>
      <CartesianCoordinates />
      <Point x={1} y={1} />
    </Mafs>
  )
}
```

```jsx live
function PointsAlongFunction() {
  const fn = (x) => (x / 2) ** 2
  const sep = useMovablePoint([1, 0], {
    constrain: "horizontal",
  })
  const points = sep.x != 0 ? range(0, 10 * sep.x, sep.x).concat(range(0, -10 * sep.x, -sep.x)) : []

  return (
    <Mafs height={500} yAxisExtent={[-1.3, 4.7]}>
      <CartesianCoordinates />

      <FunctionGraph.OfX y={fn} opacity={0.25} />
      {points.map((x, index) => (
        <Point x={x} y={fn(x)} key={index} />
      ))}
      {sep.element}
    </Mafs>
  )
}
```
